Application 100% java to download one or mores files.
It has a user friendly interface

  1. Creating links

    The links can be created by inserting the links and directories into the list and after exporting. Or creating a text file with the format:

    • [dir-name] - if you wish insert a directory ,it one must be into [] .
    • links - one or more links.
